React 설치
1. webapplication 폴더로 이동
2. 기존 frontend 폴더 삭제
3. 명령어 입력 후 설치 npx create-react-app frontend
4. 이후 터미널에서 기존 파일 덮어쓰고 설치 선택
5. 명령어로 정상 출력 되는지 확인 cd frontend, npm start

1. Spring Initializr 사이트에 접속

2. 구성 환경
Project: Gradle - Groovy
Language: Java
Spring Boot: (숫자 뒤에 SNAPSHOT이나 M이 안 붙은 가장 높은 버전 선택)
Project Metadata:
Group: com.project (원하는 이름)
Artifact: backend-spring
Java: 17 또는 21 (본인 컴에 설치된 버전)

3. 의존성 설치
Dependencies (오른쪽 ADD 버튼 클릭): 아래 4개는 필수입니다.
Spring Web: API 서버를 만들기 위해 필수
Spring Data JPA: DB 연동을 위해 필요
MariaDB Driver: MariaDB와 연결하기 위해 필요
Lombok: 코드를 깔끔하게 줄여주는 도구

4. 완성
GENERATE 버튼을 눌러 압축파일을 다운로드합니다.
압축 파일 backend-spring에 가져와서 압축해제.
